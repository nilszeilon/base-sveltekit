# base-sveltekit

this is a base repository for creating web apps with sveltekit using supabase.

```bash
#.env.local
PUBLIC_SUPABASE_URL=<your_supabase_project_url>
PUBLIC_SUPABASE_ANON_KEY=<your_supabase_anon_key>
````
